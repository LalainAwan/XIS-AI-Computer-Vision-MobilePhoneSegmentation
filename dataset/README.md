# Mobile Phone Segmentation Dataset

Dataset Summary:
- Total Images: 80
- Training Images: 56
- Validation Images: 16
- Testing Images: 8
- Annotation Format: COCO Segmentation
- Class: mobile_phone

Folder Description:
- raw/: original collected images
- annotations/: segmentation annotations
- train/: training split
- valid/: validation split
- test/: testing split

The full dataset is maintained separately and loaded during training through Google Drive.
